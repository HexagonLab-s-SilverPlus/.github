## 프로젝트 소개
**SilverPlus**는 고령화 사회의 노인 복지 증진을 목표로 하는 **AI 기반 어르신 말동무 서비스**와 **관공서 문서 작성 보조 기능**을 제공하는 통합 플랫폼입니다.  
노년층의 정서적 안정, 사회적 고립 해소, 공공서비스 접근성 향상을 지원하며, 가족 구성원의 부담을 덜고 삶의 질 향상에 기여합니다.

---

## 주요 기능

### 1. 어르신 AI 말동무 서비스
- **텍스트 및 음성 대화**: GPT-4와 gTTS를 활용하여 자연스러운 대화 경험 제공  
- **감정 분석 응답**: 대화 맥락에 따라 공감형 답변 생성  
- **실시간 음성 출력**: AI 응답을 실시간으로 음성 변환하여 제공  

### 2. 공문서 작성 보조 기능
- **질문 자동 생성 및 초안 작성**: GPT-4를 기반으로 대화형 문서 작성 지원  
- **CSV 파일 생성 및 제출 기능**: 작성된 문서를 담당자에게 자동 제출  
- **간단한 관리 페이지**: 제출 현황 및 담당자 피드백 관리  

### 3. 사용자 맞춤 인터페이스
- **어르신 친화형 UI/UX**: 큰 글씨, 명확한 대비, 단순한 네비게이션 구조  
- **가족 및 담당자 연계 기능**: 데이터 조회 및 소통 지원  

---

## 프로젝트 의의
- **디지털 소외 해소**: 노년층이 공공서비스에 접근할 수 있도록 기술적 장벽 완화  
- **정서적 지원**: AI 대화를 통한 심리적 안정 및 소통 강화  
- **공공 서비스 효율화**: 행정 문서 작성 및 제출 자동화를 통한 접근성 향상

## 기술 스택

### 프론트엔드
- React.js, Axios, CSS3  
- Figma (UI 프로토타이핑)

### 백엔드
- Spring Boot (REST API)  
- Flask (GPT-4 AI 연동 어댑터)

### 데이터베이스
- Oracle 18c

### AI / 음성 기술
- OpenAI GPT-4 (대화 및 문서 생성)  
- gTTS (Text-to-Speech 변환)

### 협업 / 관리
- GitHub (형상 관리)  
- Notion (일정 및 진척 관리)

## 주요 화면 흐름
1. **로그인 / 회원가입**  
2. **대화 화면** – AI와 자연스러운 말동무 대화  
3. **문서 작성 화면** – 대화 기반 문서 생성 및 CSV 파일 다운로드  
4. **제출 페이지** – 담당자에게 문서 자동 전송
